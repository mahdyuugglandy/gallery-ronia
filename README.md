
پکیج سایت گالری رونیا
====================

فایل‌ها:
- index.html        (صفحهٔ اصلی، شامل پنل ادمین محلی و فراخوانی `/api/tgju`)
- admin.html        (صفحهٔ نمونهٔ ادمین)
- api/tgju.js       (Serverless function برای Vercel/Netlify — می‌گیرد قیمت از TGJU)
- README.md         (این فایل)

تنظیم و استقرار در Vercel (گام‌ها):
1) یک ریپو در GitHub بساز (مثلاً ronya-gallery).
2) محتویات این پوشه را در ریپو آپلود کن.
3) به Vercel وارد شو و پروژه را Import کن (اتصال به GitHub).
4) در پنل پروژه > Settings > Environment Variables متغیرها را اضافه کن (اگر API رسمی TGJU داری):
   - TGJU_API_URL = <آدرس endpoint که TGJU بهت می‌دهد>
   - TGJU_API_KEY = <کلید API TGJU>
   سپس Redeploy کن.
5) اگر متغیرها را نگذاری، تابع `/api/tgju` مقدار نمونه را برمی‌گرداند (برای تست محلی).
6) بعد از Deploy، صفحهٔ اصلی آدرس مانند `https://<project>.vercel.app` خواهد داشت.
7) برای امنیت پنل ادمین: از Vercel Password Protection یا Netlify Identity یا Cloudflare Access استفاده کن (رمز داخلی در فایل index.html فقط برای تست محلی است).

توجه فنی مهم درباره TGJU:
- ساختار JSON پاسخ TGJU ممکن است متفاوت باشد. بعد از دریافت کلید و URL، تابع `api/tgju.js` را مطابق ساختار پاسخ TGJU اصلاح کن تا فیلدهای `gold_18_per_gram` و `coin_full` را صحیح استخراج کند.
- به احتمال زیاد TGJU محدودیت نرخ (rate limit) دارد؛ درخواست هر 1 ثانیه ممکن است منجر به بلاک شدن شود. پیشنهاد می‌شود در قرارداد API مشخص کنید که چه فرکانسی مجاز است؛ در کد فرانت‌اند مقدار به‌صورت پیش‌فرض روی 5000ms قرار دارد.
